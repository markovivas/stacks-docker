# Portainer CE

Interface web para gerenciar containers Docker. Gerencia todos os containers do host, independente de como foram criados.

Acesso: [https://localhost:9443](https://localhost:9443)

---

## Recursos

| Recurso          | Detalhe                    |
| ---------------- | -------------------------- |
| Web (HTTPS)      | https://localhost:9443     |
| Web (HTTP)       | http://localhost:9000      |
| Container        | `portainer`                |
| Dados            | `portainer_data` (volume)  |

---

## Primeiro acesso

1. Acesse https://localhost:9443
2. Crie o usuário administrador
3. Selecione "Get Started" para conectar ao Docker local

## Funcionalidades

- Parar, iniciar, reiniciar e remover containers
- Atualizar imagens Docker
- Visualizar logs em tempo real
- Gerenciar redes, volumes e imagens
- Criar containers e stacks (compose) pela interface
- Monitorar uso de CPU e memória
