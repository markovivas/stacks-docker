# Navidrome

Servidor de streaming de música auto-hospedado. Suporta diversos formatos de áudio (MP3, FLAC, AAC, etc.) e tem interface web para ouvir de qualquer dispositivo.

Acesso: [http://localhost:4533](http://localhost:4533)

---

## Recursos

| Recurso     | Detalhe                   |
| ----------- | ------------------------- |
| Web         | http://localhost:4533     |
| Container   | `navidrome`               |
| Músicas     | configure no compose      |
| Dados       | `./data/`                 |

---

## Primeiro acesso

1. Acesse http://localhost:4533
2. Crie o usuário administrador
3. Configure o caminho da pasta de músicas

## Suporte a formatos

- MP3, FLAC, AAC, OGG, WMA
- WAV, AIFF, ALAC
- E outros formatos comuns
