# Stirling PDF

Ferramenta web para realizar diversas operações em arquivos PDF: divisão, união, conversão, compressão, OCR e muito mais.

Acesso: [http://localhost:8090](http://localhost:8090)

---

## Recursos

| Recurso           | Detalhe                     |
| ----------------- | --------------------------- |
| Web               | http://localhost:8090       |
| Container         | `stirling-pdf`              |

---

## Funcionalidades

- Dividir / Unir PDFs
- Converter PDF para Word, Excel, imagens
- Comprimir PDFs
- OCR em documentos escaneados
- Extrair / remover / rotacionar páginas
- Assinar digitalmente
- Adicionar números de página, marcas d'água
- E muito mais...
